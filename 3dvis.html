3D Visualization of the Infrastructure on Sable Island with LiDAR Data

Introduction:

The purpose of this project was to create a 3D scene of the infrastructure on the Sable Island Natioanl Park Reserve. To begin, LiDAR data was downloaded from the Province of Nova Scotiaâ€™s Data Locator - Elevation Explorer and brought into ArcGIS Pro for processing. Then, the footprints of the structures were identified by filtering the LiDAR, and converting the data to polygons. The polygons were edited to match the structures as they appeared on GeoNOVA's 10k orthophoto, which was used as a basemap. Next, elevation features had to be created, including a digital elevation model (DEM), a digital surface model (DSM), as well as a normalized raster. These elevation features were used to determine the height of structures on the Island. The structures and roofs were created and symbolized utilizing ArcGIS Online's 3D Basemaps Solution. The data was then converted to multipatch, and the textures were downloaded from the free, open source, and copyright-free website Texture Ninja and added to the scene.

Geoprocessing Tools Used:

  • LAS Point Statistics As Raster
  • Raster to Polygon
  • Eliminate Polygon Part
  • Regularize Building Footprint
  • LAS Dataset To Raster
  • Minus
  • Extract Roof Form (3D Basemaps Solution)
  • Apply Symbology From Layer
  • LOD2BuildingShells_meters.lyrx (3D Basemaps Solution)
  • Layer 3D to Feature Class
